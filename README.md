## Cara Instal dan Jalankan di Termux
1.Install pkg dulu
```bash
pkg update && pkg upgrade
pkg install python
```

2. Clone repo ini:
```bash
git clone https://github.com/kamunanay/encrypt-decrypt-ganz.git
cd encrypt-decrypt-ganzmods
```


## Jalankan Programnya 
3.Jalankan Kode nya
```bash
python encrypt_decrypt_ganzmods.py
```
